Considerações gerais
Utilizar GSAP e ScrollTrigger para animações de entrada e controle de scroll
Utilizar Framer Motion para interações locais como hover e animações de botões e cards
Utilizar Lottie para substituir ícones estáticos por microanimações
Garantir que as animações sejam suaves e não excessivas
Manter a performance da página alta com lazy load de elementos pesados como Lottie
Utilizar máscara linear gradient para criar efeitos de fade nas bordas onde necessário
Adicionar transições CSS simples para microinterações quando possível para manter a leveza
